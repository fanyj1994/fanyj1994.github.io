《江城》：熟悉又陌生的景致

我是从重庆乘慢船顺江而下来到涪陵的。那是1996年8月底一个温热而清朗的夜晚——长江上空星斗闪烁，漆黑的水面却映不出微弱的点点星光。

我在上大学时就买到了《江城》，到今天才看完。之所以今天才看完，是因为八月份，我从网上偶然看到何伟回到成都，在川大任教的消息，才想起来这本书已经积灰很久。

现在我读完的结论是，这是一部十分优美的纪实作品，我给五颗星。这本书是那种乍一看并不喜欢，读过去会突然发现，我需要的正是这种感觉，一种生活细微处的认真与安然。

这种感觉指的是，作为一个外国人，他将我熟悉又陌生的中国细致地罗列出来。熟悉指的是，涪陵的乱糟糟的街头巷尾，嘈杂拥挤的人群，害羞内敛的学生，我都曾置身其中，走过无数次，但在我的记忆中，这些场景长久地隐藏在一团迷雾之后，哪怕行走在其中，也呼唤不出来，何伟的文字，将这些记忆拉出来，像一幅画卷一样徐徐展开，里面的人物，都赋予一个广角镜头，动弹起来，那些味道，声音，也都跟着跳了出来，噢，就是这种感觉！

不知道你还记不记得罗中立的油画作品《父亲》，那个面庞黢黑，额上沟壑纵横的形象，是多么常见，但那一道道皱纹有多深，几乎没人注意过。而何伟正将这一道道属于涪陵的皱纹清晰地娓娓勾勒出来。

我讲其优美，是因为作为一个外国人笔下的中国，其并不像游记，用一些精饰的笔法向美国人浓墨重彩渲染东方之神韵。也不用做一个旁观者，刻意去保持客观，用冷峻的笔触记录中国，或者使用描述好山好水的风情万种的优雅词汇，他是一个亲历者，将自己努力地融入到中国，并不刻意寻找一个外国人的视角，也不存在隔阂，他很快学会了独特的中国生活节奏，学会与中国人交谈，把握中国人的性格。他慢悠悠地摇过灰尘横飞的街头，同街边晒太阳的老人交谈，和好奇的孩子打招呼，观察城市边缘的石梁。这样一种流水般自然的节奏，带着优美的韵律。

而作为一个中国人，我觉得自己并没有这种舒逸的节奏。面对熟悉的土地，总是像过客一般匆匆而过，彼此经过的时候，粗粗地打量一眼，便错身而过，我更认为现代人走在一种错位之中，从来没有感受到脚下土地的存在。而本书，让我无数次捕捉到切实站在这片国度上的真实之感：看，这就是中国。

作为一本纪实文学，它的叙述笔触，这正是我期待的感觉。作为对待生活的态度，何伟忘记了岁月，哪怕过了二十三年的今天，你看到他笑眯眯地出现在成都的街头，好像，生活就在他的脚下延伸出来。另外，我很喜欢这本书的翻译，十分优美。虽然我没有读过英文原版，但李雪顺的译文，已足够精彩。
